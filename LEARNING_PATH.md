## 📅 学习计划（建议 3-5 天完成）

### 第 1 天：基础理论与第一阶段

- [ ] 阅读 `README.md` 了解项目概况
- [ ] 阅读 `QUICKSTART.md` 快速入门
- [ ] 学习基础概念：
  - [ ] 什么是线程池？
  - [ ] 生产者-消费者模式
  - [ ] `std::thread` 基本用法
  - [ ] `std::mutex` 和 `std::lock_guard`
  - [ ] `std::condition_variable`

  - [ ] 异常传播机制

- [ ] 理解模板函数和完美转发
- [ ] 阅读 `src/step2_future/main.cpp`
- [ ] 运行 `./step2_with_future`
- [ ] 练习：提交带返回值的任务
- [ ] 练习：批量任务处理
- [ ] 练习：异常处理

---

### 第 3 天：高级特性与第三阶段

- [ ] 学习高级概念：
  - [ ] `std::priority_queue`
  - [ ] `std::atomic` 原子操作
  - [ ] RAII 设计模式
  - [ ] 异常安全保证
- [ ] 阅读 `src/step3_advanced/ThreadPool.h`
- [ ] 理解优先级队列的实现
- [ ] 阅读 `src/step3_advanced/main.cpp`
- [ ] 运行 `./step3_advanced`
- [ ] 练习：使用不同优先级
- [ ] 练习：查看性能统计
